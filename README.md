A [Flarum](http://flarum.org) extension. a flarum language translator

git tag v0.0.0
git push origin v0.0.0


## Installation

Install with composer:

```sh
composer require dhtml/translate:"*"
```

## Updating

```sh
composer update dhtml/translate:"*"
php flarum migrate
php flarum cache:clear
```
